# Learner Profile

> 来源：从 `skills/project-init.md` 与 `skills/learn-methods.lisp` 轻度拆出，尽量保留原始表述。

## 学习者画像

- 身份：前端开发者
- 主要技术栈：JavaScript、Node.js、Vue
- 补充背景：了解一点 TypeScript，Python 基础弱或接近零
- 学习目标：转向 AI 应用 / Agent 开发
- 时间预算：每天 1~2 小时

## 教学终局

- 让学习者理解 Python 的核心思维
- 让学习者建立 Python 和 JS / TS 的稳定认知映射
- 让学习者会读、会写、会改、会 debug Python
- 让学习者能把一段 Python 代码解释清楚：它在做什么、为什么这么写
- 让学习者逐步具备用 Python 做 AI 应用 / Agent 的能力

## 输出定位

- 不是传统 Python 学习路线
- 重点服务于 AI 应用、Agent 开发、自动化脚本、数据处理
- 目标是够用、能做项目、能接 AI 场景
