# Day 7 Practice Package Completion Design

## Goal

Complete the Day 7 daily package by adding the missing standalone exercise file and linking it from the lesson.

## Scope

Files to change:

- `study/week3/day7/practice.py`
- `study/week3/day7/lesson.md`

## Design

Day 7 already works as a project day, so the practice file should reinforce the same project rather than introducing unrelated toy exercises.

The practice file will include:

- file header and run instructions
- explicit list of the new content it uses
- minimal example area using the Day 7 helper functions
- exercise 1: warm-up / read-output / fill-blank style
- exercise 2: coding task with `TODO`
- exercise 3: bug explanation / debugging task
- answer area
- check standards
- AI review instructions

The lesson update will be minimal:

- add one clear line pointing learners to `practice.py`
- make it obvious that Day 7 now has both project code and a separate exercise file

## Constraints

- Use only Week 3 knowledge already introduced.
- Keep the file runnable as a script.
- Keep the exercises focused on the actual Day 7 project flow.
