# Python基础 / 5.1 Mistakes

## 今天最容易出现的错误

### 错误 1：布尔值和空值大小写写错

- 现象：把 `True` 写成 `true`，把 `None` 写成 `null`
- 原因：直接套用了 JS / TS 的字面量习惯
- 修复动作：把 Python 的基础值字面量单独记一遍，尤其是 `True`、`False`、`None`

### 错误 2：字符串和数字直接相加

- 现象：写出 `"score=" + 95` 之类的表达式
- 原因：把 JS 的隐式拼接经验直接带进 Python
- 修复动作：优先改成 f-string，或者先显式转字符串
