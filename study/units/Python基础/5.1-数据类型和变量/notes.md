# Python基础 / 5.1 Notes

## 这次新记住的点

- Python 变量是“名字绑定值”，不是先声明固定类型
- `None` 更接近 JS 的 `null`，但 Python 没有 `undefined` 这个值
- Python 里 `True` / `False` 首字母必须大写
- 字符串和数字不要直接用 `+` 混拼，优先用 f-string

## 和 JS / TS 的对照

- `let score = 95` 对应 `score = 95`
- `null` 对应 `None`
- 未声明变量在 JS 里常想到 `undefined`，但 Python 会直接报错
