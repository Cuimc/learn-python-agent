# Study Templates

本目录提供 `study/units/<大单元>/<小单元>/` 的可选索引模板。

`study/<day-id>/` 不走这里的模板，而是由 `skills/templates/day-plan-template.md` 和 `scripts/init_day_plan.py` 生成。

## 使用方式

- 新开一个学习单元时，只生成索引文件
- 小单元目录建议使用：`<outline-id>-<中文标题>`
- 索引信息可指向固定大纲和资料来源
- 不要把这里的内容当成当天学习任务输出

## 标准文件

- `unit.json`
- `refs.md`
