# Study Templates

本目录提供 `study/units/<大单元>/<小单元>/` 的标准模板。

`study/dayN/` 不走这里的模板，改为由 `skills/templates/day/` 和 `scripts/init_day_plan.py` 生成。

## 使用方式

- 新开一个学习单元时，复制 `unit/` 下的文件结构
- 小单元目录建议使用：`<outline-id>-<中文标题>`
- 再根据当前固定大纲节点和动态大纲状态替换占位内容
- 不要把模板内容直接当成正式学习产出

## 标准文件

- `lesson.md`
- `practice.py`
- `answers.md`
- `notes.md`
- `mistakes.md`
- `unit.json`
